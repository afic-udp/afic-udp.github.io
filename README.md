# Ayudantías FIC

Sitio web de AFIC con materiales de apoyo para estudiantes de la Facultad de Ingeniería y Ciencias de la Universidad Diego Portales.

Sitio web: https://afic-udp.github.io/

## Documentación

- [Cómo subir materiales](./docs/CONTRIBUTING.md)
- [Reglas de naming](./docs/NAMING.md)
- [Cursos y códigos](./docs/COURSES.md)

## Desarrollo

```bash
npm install
npm run dev
npm run check
```

Los cambios se suben por pull request hacia `main`. El sitio se publica automáticamente después del merge.
